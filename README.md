# Calculator

> 드디어 UI가 생긴 레모니표 계산기 앱입니다! 연산의 정교함은 떨어졌지만, 떨어진 만큼의 에너지를 UI구현에 쏟았답니다!  

---

## 📋 프로젝트 개요

> 내배캠 프로그래밍 주차를 맞이하여 스타터 개인 과제로 주어진 <계산기 만들기>를 수행하며 시작된, 세상에 단 하나밖에 없는 UI 없는 계산기가 드디어 앱 개발 입문 주차를 맞이하야 UI가 있는 계산기로 거듭났습니다!!

### 대상 사용자

- 내배캠 과제를 찍먹해 보고 싶은 사람
- UI 사용을 선호하는 사람  
- 레모니가 짠 코드를 돌려보고 싶은 사람 
---

## 📱 주요 기능

1. **더하기 빼기 곱하기 나누기 나머지**  
   기본적인 사칙연산 및 나머지 계산이 가능합니다.

2. **터치**  
   눈으로 보고 터치할 수 있습니다.
